# Legal and Compliance

This section contains legal and compliance information for operators,
users, contributors, rights holders, and representatives.

Use this page as the starting point:

- Operators and users: start with the
  [Legal Notice](/legal/legal-notice) and
  [Acceptable Use](/legal/acceptable-use).
- Contributors: read
  [Contributor IP and Submission Rules](/legal/contributor-ip).
- Rights holders or representatives: start with
  [DMCA and Rights-Holder Contact](/legal/dmca), then review the
  [Rights-Holder Notice](/legal/rights-holder-notice) if the concern is about
  repository-controlled material.

## Quick Boundary Summary

AniBridge is open source software. It does not host or operate third-party
catalogs, third-party media, or independent deployments run by users.

Repository maintainers can review repository-controlled materials such as:

- source code;
- documentation;
- repository assets;
- official release materials published from this repository.

Repository maintainers cannot directly remove or disable:

- third-party hosted media;
- independent deployments;
- third-party portals, hosters, CDNs, or external streaming infrastructure.
